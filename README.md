# OpenSQLManager

OpenSQLManager is an open source software maked in Python 3 and Tkinter for Manage SQL Databases.

## Features
* Connect to Database
* Run queries
* Import SQL scripts
* Export SQL scripts
* Multiple language support
* Cross plataform

## Install
```bash
sudo apt-get install python3-tk
```

## Run
```bash
python3 __init__.py
```

## Configuration
src/config.yaml
